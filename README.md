## Project Overview
A simple, interactive command-line calculator built with Python to demonstrate core programming logic. 

## Key Features & Logic
* **User Input Handling:** Collects numbers and operators dynamically from the user.
* **Conditional Logic:** Uses `if`, `elif`, and `else` statements to control the program flow based on the chosen operation.
* **Supported Operations:** Performs basic arithmetic (+, -, *, /) as well as exponentiation (**).
* **Error Handling:** Safely catches invalid inputs and operations to prevent crashes.

## Purpose
Developed as a foundational project to master Python basics, clean code structure, and problem-solving through terminal applications.
